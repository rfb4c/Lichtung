## 🎯 项目定位

林间空地是一个**研究用 Demo**，旨在验证两种干预路径对感知极化的校准效果：

- **路径 B**：通过可视化真实民调数据，修正用户对公众态度分布的感知
- **路径 C**：通过展示交叉身份标签，软化政治对立群体间的边界

本项目不是完整的新闻产品，而是用于实验验证的概念原型。
