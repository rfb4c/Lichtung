## 待实施设计决策（已讨论，尚未开发）

### 右侧栏重设计（讨论日期：2026-03-10）

**背景**：原右侧栏完全模仿 Twitter（热门话题、关注推荐），所有元素均无实际功能，与研究目标无关。

**决策结果**：

| 元素 | 决策 | 说明 |
|---|---|---|
| 搜索框 | ✅ 保留并实现 | 按 Topic 名称搜索，回车确认，筛选主 Feed |
| 热门话题卡片 | ❌ 移除 | 与研究目标相悖，不应强化已有热点 |
| 关注推荐 / Follow 按钮 | ❌ 移除 | 与平台定位无关 |
| Show more | ❌ 移除 | 无内容支撑 |
| **冷门话题入口** | 🔜 占位保留，未来实现 | 见下方 |

**搜索交互规范**：
- 搜索对象：`topic.name`（议题名称，如 "Gun Control"）
- 触发方式：按 Enter 确认
- 效果：主 Feed 只显示该 Topic 下的报道
- 重置方式：点击侧边栏 Home 按钮回到全部 Feed（无需额外"清除"按钮）

**冷门话题（未来功能）**：
- 设计动机：主动引导用户接触关注度较低的议题，增加信息多样性，符合降低感知极化的研究目标
- 定义待定：候选方案为"评论数/报道数最少的 Topic"，或手动标记
- 交互方式：点击后与搜索筛选效果相同（筛选 Feed）
- **当前阶段**：仅保留 UI 入口（占位卡片），不实现筛选逻辑

### FeedItem 动作按钮重设计（讨论日期：2026-03-10）

**背景**：原动作栏完全模仿 Twitter，转发/点赞/浏览量显示伪随机数字，全部无功能。

**各按钮决策**：

| 按钮 | 决策 | 说明 |
|---|---|---|
| 评论 | ✅ 保留（已实现） | 展开评论区，计数真实 |
| 转发 | ❌ 移除 | 平台无发帖功能，无意义 |
| 点赞 | ❌ 移除 | 伪随机数字，无实际功能 |
| 浏览量 | ❌ 移除 | 伪随机数字，无实际功能 |
| ··· 更多菜单 | ❌ 移除 | 无内容支撑 |
| **收藏** | 🔜 保留按钮，暂不实现 | 与 Bookmarks 功能联动，未来实现 |
| **分享** | ✅ 实现（思路 A） | 见下方规范 |

**分享功能规范**：
- 分享对象：报道的**原始外部链接**（`report.url`），不是平台内链接
- 实现方式：优先调用 `navigator.share`（移动端系统分享菜单）；不支持时 fallback 到复制链接到剪贴板 + toast 提示"链接已复制"
- 前置条件：`report.url` 必须是有效的外部链接（静态数据需确认）

### 未来功能规划（讨论日期：2026-03-10）

以下功能已形成方向共识，暂不实现，仅作规划记录：

**Bookmarks（收藏报道）**
- 研究价值：用户收藏内容可反映信息偏食程度，是衡量信息茧房的潜在数据来源
- 在 Profile 页展示"收藏的报道"列表，按 Topic 分组

**Following → 关注 Topic**
- 重定位：不关注人，而是关注议题（Topic）
- 研究价值：用户主动订阅议题的选择行为，可作为干预效果的测量指标
- 实现后影响 Feed 排序：关注的 Topic 优先展示

**Notifications（通知）**
- 与评论回复功能联动（前置依赖：嵌套评论）
- 触发场景：有人回复了你的评论

**评论回复功能（嵌套评论）**
- 目前只有顶层评论，未来实现嵌套回复，是 Notifications 的前置依赖

**信息饮食透明度（Information Diet）**
- 研究价值：**高**，直属感知极化干预范畴，有文献支持（元认知干预）
- 展示用户本次访问浏览了哪些 Topic 的报道，如"你今天浏览了 Gun Control (5篇)、Climate Policy (2篇)"
- 位置候选：右侧栏下方 / Profile 页
- 实现前置条件：需要浏览行为记录（前端 sessionStorage 或 Supabase 行为日志）

**Explore → 议题探索页**
- 研究价值：**中-高**，鼓励用户主动接触不熟悉的议题
- 展示所有 Topic 的卡片列表，每张卡片显示议题名、报道数、简介
- 点击进入该 Topic 的筛选 Feed

**先评论后看评论（Comment-First）**
- 研究价值：**高**，成本**低**
- 用户展开评论区时，先显示输入框；点击"查看评论"后才加载其他人的评论
- 减少从众效应，帮助用户形成独立判断，直接对抗评论区回音壁造成的感知极化

**Discover New Topics（冷门话题）**
- 右侧栏占位已实现，筛选逻辑待实现
