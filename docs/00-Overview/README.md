# 00-Overview - 项目总览

本文件夹包含「林间空地」项目的核心设计文档。

## 📄 文档列表

### [产品设计文档.md](./产品设计文档.md)
**阅读对象**：所有参与者
**内容概要**：
- 研究背景：多数错觉（Majority Illusion）与感知极化
- 干预假设：可视化真实分布如何修正用户感知
- 核心功能：路径 B（共识可视化）和路径 C（交叉身份）
- 数据需求：民调数据、报道采集、用户标签

### [技术设计文档.md](./技术设计文档.md)
**阅读对象**：开发者
**内容概要**：
- 系统架构：Vite + React + Supabase
- 数据库设计：events、reports、profiles、comments 表结构
- 前端改动范围：组件设计、API 调用
- 实施步骤：从 Supabase 配置到数据导入

### [设计更新日志.md](./设计更新日志.md)
**阅读对象**：追踪项目进度的所有人
**内容概要**：
- 语义化版本号记录（v0.1.0, v0.2.0, v0.2.1, ...）
- 每次迭代的功能变更、文件修改、数据库变更
- 当前状态快照与下一步计划

## 🔄 文档关系

```
产品设计文档（为什么做？做什么？）
        ↓
技术设计文档（怎么做？用什么技术？）
        ↓
设计更新日志（做到哪里了？）
```

## 📌 使用建议

1. **首次阅读**：按顺序读完三份文档，建立完整认知框架
2. **后续查阅**：
   - 需要理解设计意图 → 产品设计文档
   - 需要实现细节 → 技术设计文档
   - 需要了解进度 → 设计更新日志
